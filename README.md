### zsh

#### 相关文件

- [.zshrc](.zshrc)

#### 说明

- 文件放在根目录`~`下。
- 添加了`numf`和`numd`命令，分别可以统计当前文件夹下的`文件数`和`文件夹数`。
- 支持`ll`命令。
- 添加快捷键`ctrl+f`补全命令，`ctrl+j`执行命令(配合oh-my-zsh使用)。

---
